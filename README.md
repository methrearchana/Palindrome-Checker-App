### Palindrome Checker Application...###

1. OVERVIEW :
    A simple and interactive Palindrome Checker built using HTML, CSS, and JavaScript.
    This application allows users to enter any word, phrase, or number and instantly check whether it is a palindrome.

2. TECHNOLOGY USED :
   
   HTML5 – Structure
   
   CSS3 – Styling and layout
   
   JavaScript (ES6) – Palindrome logic and DOM handling.
